# **App Name**: ZarRice

## Core Features:

- Product Catalog: Display premium Iranian rice varieties (Tarom, Sadri, Hashemi, Fereydunkenar, Gilan, Golestan) with images, descriptions, weight options, and pricing.
- Shopping Cart & Checkout: Implement a shopping cart system with checkout, order tracking, and secure payment integration.
- User Accounts: Allow users to create and manage accounts, view order history, and save addresses.
- Blog Section: Publish articles related to rice cultivation, cooking tips, and storage guides to improve user engagement.
- Contact & About Pages: Include 'Contact Us' page with a form, phone, email, Google Maps, and an 'About Us' page with company story and trust badges.
- Language toggle and Translation: Use a translation tool for text localization. Serve text depending on selected language.

## Style Guidelines:

- Primary color: Saturated gold (#D4AF37) to represent the premium quality of the rice and Iranian heritage.
- Background color: Very light gold (#FAF7EF) to provide a bright, inviting, and luxurious feel.
- Accent color: Burnt orange (#CC7722), an analogous color, to create contrast and highlight important elements like call-to-action buttons.
- Body text: 'Inter', a grotesque-style sans-serif. Headline Text: 'Space Grotesk', a proportional sans-serif font with a computerized, techy, scientific feel; good match with the body text font, suitable for headlines and short amounts of body text.
- Employ a glassmorphism design with semi-transparent panels, frosted glass effects, and subtle gradient overlays to create a modern and visually appealing interface.
- Implement a dark/light mode toggle for user preference.
- Use RTL (Right-to-Left) layout for Persian language support.
- Use smooth animations for transitions and interactive elements.
